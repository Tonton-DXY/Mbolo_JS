# Saison 1 : Mbolo JS

## Objectif :
Se familiariser avec la syntaxe de Javascript et typeScript.

## INTRODUCTION GENERALE
Le but de cette saison est de prendre progressivement la main sur le Javascript.
Ici, l'utilisation de bibliothèques JavaScript est proscrit.
Chaque exercice est un épisode de la saison et donne lieu à la création d’un nouveau script.

# Les épisodes :
épisode 1: Écrire un programme qui déclare une variable de type chaîne de caractères contenant le message « Mbolo JS ».

épisode 2: Écrire un programme qui demande le nom de l'utilisateur et affiche une alert "Bonjour" suivis du nom de l'utilisateur.

épisode 3: Écrire un programme qui demande à l'utilisateur deux nombres et affiche leur somme en alerte.

épisode 4: Écrire un programme permettant de permuter les valeurs de deux variables. Le résultat est affiché en console.

épisode 5: Écrire un programme qui demande à l'utilisateur un nombre et affiche son double en alerte.

épisode 6: Écrire un programme qui demande à l’utilisateur trois nombres et affiche la moyenne de ces nombres en alerte.

épisode 7: Écrire un programme qui demande l'âge de l’utilisateur et affiche : Vous êtes mineur ou Vous êtes majeur.

épisode 8: Ecrire un programme qui demande le nom de l’utilisateur et son sexe.

épisode 9: Écrire un programme qui demande la moyenne d’un élève et affiche sa mention.

épisode 10: Écrire un programme qui applique une remise de 10% sur une facture de plus de 40.000F

épisode 11: Écrire un programme qui demande un nombre à l’utilisateur et affiche la table de multiplication de ce nombre   jusqu’à 10.

épisode 12: Écrire un programme qui fait la somme des 10 premiers nombres entiers à partir de 1. Le résultat est affiché en console.

épisode 13: Écrire un programme qui demande sans cesse à l'utilisateur de renseigner la bonne réponse d’une opération.

L'utilisation de Vite.dev est recommandé pour initialiser le projet.
